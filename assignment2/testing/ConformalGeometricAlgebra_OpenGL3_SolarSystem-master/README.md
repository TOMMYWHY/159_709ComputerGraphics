# Systeme-Solaire-C3GA

### Prerequisites
* `CMake >= 3.7`
* `SDL2 >= 1.2`
* `OpenGL >= 3.3`
* `GLEW >= 2.1.0`
* `eigen3 >= 3.3.0


### Dependencies
Garamon -> we used a **CGA** C++ library generated through [Garamon](https://github.com/vincentnozick/garamon)


### Compilation
The compilation is done through `CMake` :
````bash
mkdir build
cd build
cmake ../SolarSystem
make
